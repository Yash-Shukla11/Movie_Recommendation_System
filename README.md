<h2 align="center"> 📋 Overview </h2>

<p align="center">
  This application uses Machine Learning to analyze movie metadata—including genre, cast, and crew—to provide personalized movie recommendations. By selecting a movie you enjoy, the system finds the most similar titles in the database.
</p>

<h2 align="center"> ✨ Key Features </h2>

*   **Interactive Search:** Instantly find your favorite movies from the database.
*   **Intelligent Recommendations:** Uses cosine similarity to find the most relevant matches.
*   **Visual Interface:** Clean, responsive design featuring movie posters and descriptions.
*   **Deployment Ready:** Hosted via Streamlit Cloud for 24/7 access.

<h2 align="center"> 💻 How to Run Locally </h2>
If you want to run this project on your own computer, follow these steps:

1. **Clone the repository:**
   `git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git`

2. **Install required packages:**
   `pip install -r requirements.txt`

3. **Run the application:**
   `streamlit run app.py`
